# Chapitre 2 : La Caisse et l'Encaissement

La caisse est le cœur financier de votre agence sur Gestion360. C'est ici que vous suivez tous les flux monétaires : encaissements de loyers, paiements des fournisseurs, et reversements aux propriétaires.

Une bonne tenue de caisse est la garantie d'une comptabilité saine et de propriétaires satisfaits.

---

## 1. Comprendre le Journal de Caisse

Le journal de caisse est votre tableau de bord quotidien. Il trace chaque mouvement d'argent.

*   **Où aller ?** : Menu **Caisse** > *Journal des transactions*.
*   **Ce que vous y trouverez** :
    *   **Les Entrées (Crédits)** : Les loyers payés par les locataires, les frais d'agence encaissés, etc.
    *   **Les Sorties (Débits)** : Les reversements (payouts) aux propriétaires, le paiement d'un plombier suite à un ticket d'incident, etc.
    *   **Le Solde Global** : L'argent total disponible sur le compte de l'agence à l'instant T.

> [!NOTE]
> Chaque transaction est horodatée et liée à un utilisateur (qui a saisi l'opération ?), ce qui garantit une traçabilité totale (Audit Trail).

---

## 2. Encaisser un Loyer (Quittance)

L'action la plus courante est l'encaissement d'un loyer. Une fois qu'un contrat de bail est actif (voir Chapitre 1), les appels de loyer se génèrent automatiquement.

*   **Où aller ?** : Menu **Quittances / Recouvrement**.
*   **Ce qu'il faut faire** :
    1.  Repérez la quittance en attente de paiement (statut "Impayé" ou "Partiel").
    2.  Cliquez sur le bouton **Encaisser**.
    3.  Indiquez le **montant réellement perçu** (il peut être inférieur au loyer prévu si le locataire paie en plusieurs fois).
    4.  Choisissez le **Moyen de paiement** (Virement, Espèces, Chèque, Mobile Money).
    5.  Validez.
*   **🎉 Résultat magique** :
    *   Le statut de la quittance passe à "Payé".
    *   Le PDF de la quittance est généré et envoyé au locataire.
    *   L'argent apparaît instantanément dans votre Journal de Caisse.
    *   Le compte du propriétaire est automatiquement crédité (moins votre commission d'agence !).

---

## 3. Les Saisies Manuelles (Dépenses Imprévues)

Parfois, vous devez sortir de l'argent de la caisse pour une dépense qui n'est pas un reversement propriétaire (ex: Achat de fournitures, frais bancaires).

*   **Où aller ?** : Menu **Caisse** > *Nouvelle Transaction*.
*   **Ce qu'il faut faire** :
    1.  Choisissez le type : **Débit** (Sortie d'argent).
    2.  Saisissez le montant.
    3.  Sélectionnez une catégorie (ex: "Frais généraux", "Entretien").
    4.  *(Optionnel)* : Si cette dépense est liée à un bien spécifique, liez-la à la résidence concernée pour impacter la rentabilité du bien.

---

## 4. La Sécurité Avant Tout (Soft Delete)

Dans Gestion360, la sécurité financière est primordiale. Vous remarquerez que **vous ne pouvez pas supprimer** une transaction par erreur. 

Pour éviter les "trous" dans la comptabilité :
1.  Seuls le **Directeur** et le **Chef d'agence** ont les droits pour annuler une transaction (si l'option de sécurité est activée dans les paramètres).
2.  Une transaction annulée n'est pas effacée, elle est "Barrée" (ou archivée) avec le nom de la personne l'ayant annulée. 

> [!TIP]
> **En résumé pour la Caisse :**
> Laissez le logiciel travailler pour vous ! N'inventez pas de transactions manuelles pour les loyers, passez toujours par l'encaissement des quittances. C'est la garantie que l'argent est correctement affecté (Agence, Propriétaire, Locataire).
