## 🛠️ Pourquoi l'historique a-t-il disparu dans l'interface ?
Lors de la mise à jour d'Antigravity, un nouveau système d'organisation des conversations par **Projets** a été introduit. Dans votre configuration locale (`antigravity_state.pbtxt`), le statut de migration est resté bloqué en cours :
```pbtxt
migrate_convos_into_projects: MIGRATION_STATUS_STARTED
```
Pendant que la migration est active, l'interface graphique cherche les conversations dans le nouveau format projet et peut masquer temporairement les anciennes conversations non encore indexées. Ce fichier index vous sert de passerelle directe pour ne rien perdre pendant que le système se stabilise.
